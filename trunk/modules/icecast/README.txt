The icecast project provides a namespace for drupal modules that 
integrate with the Icecast free streaming media server.  Bug 
reports, feature requests, patches, and new contributions are 
welcome and encouraged.

For details, please see the module-specific README.txt files.

WWW: http://drupal.org/project/icecast
